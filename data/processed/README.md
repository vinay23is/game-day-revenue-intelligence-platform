# data/processed/

Cleaned CSVs and model-ready feature datasets written by the pipeline:
- `teams.csv`, `games.csv`, `attendance.csv`, etc. — cleaned versions
- `features_attendance.csv` — model features for attendance prediction
- `features_revenue.csv` — model features for revenue prediction
- `features_full.csv` — full merged feature dataset
- `attendance_predictions.csv` — model predictions vs actuals
- `revenue_predictions.csv` — revenue model predictions
- `fan_profiles_segmented.csv` — fan profiles with K-Means cluster labels
